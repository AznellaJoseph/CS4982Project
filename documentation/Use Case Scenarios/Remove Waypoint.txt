Use Case Scenario: Remove Waypoint

Context: The user is logged in and has selected a trip

Primary Flow:
1. The user selects a date on the calendar in the range of the trip
2. The user selects the "X" button on a waypoint from the list of events for that date
3. The user confirms the removal of the waypoint from the trip

Alternative Flows:

The user cancels the remove operation
A3. The user does not confirm the removal of the waypoint
A4. The user is redirected to the list of waypoints for the chosen date
